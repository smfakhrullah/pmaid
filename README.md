# 📱 PMA IDigital - Sistem Pengurusan Sekolah Pintar

Laman web rasmi dan dokumentasi interaktif untuk **PMA IDigital**, sebuah sistem pengurusan sekolah bersepadu yang dibangunkan menggunakan **Google AppSheet** untuk **Pondok Moden Al-'Abaqirah (PMA)**.

Ekosistem ini direka khas bagi mengurangkan beban kerja manual guru dan staf melalui:

- 🤖 Automasi pintar
- 🧠 Integrasi Kecerdasan Buatan (AI)
- 📄 Penjanaan laporan PDF automatik
- 🔔 Push Notification masa nyata

---

# 📂 Struktur Repositori

Repositori ini mengandungi fail-fail berikut:

| Fail / Folder | Keterangan |
|---------------|------------|
| `index.html` | Laman web rasmi (Landing Page) PMA IDigital untuk hebahan umum, muat turun dan rujukan modul. |
| `user_guide.html` | Panduan pengguna premium (27 slaid) lengkap dengan navigasi dan tangkap layar aplikasi. |
| `pic/` | Menyimpan logo, gambar peranti dan screenshot aplikasi PMA IDigital. |

---

# 🚀 Pemasangan & Akses Aplikasi

## 📱 1. Telefon Pintar (Android & iOS)

Sesuai untuk:

- Push Notification
- Penggunaan harian
- Komunikasi staf
- Akses pantas aplikasi

### Langkah Pemasangan

1. Muat turun aplikasi **AppSheet** daripada Google Play Store atau Apple App Store.
2. Klik pautan pemasangan **Install PMA IDigital on Mobile**.
3. Selepas aplikasi dibuka, pilih **Add to Home Screen** supaya ikon aplikasi berada di skrin utama telefon.

---

## 💻 2. PC / Laptop / Tablet

Sesuai untuk:

- Pentadbiran
- Kemas kini data
- Penjanaan laporan
- Muat turun PDF

### Cara Akses

Buka pautan:

> **PMA IDigital (Browser Version)**

**Nota**

Tiada pemasangan diperlukan. Hanya log masuk menggunakan akaun rasmi PMA.
